# FLYERSDIGITALES

Sitio web que presenta servicios digitales de marketing y branding para comercios locales.
Incluye una página principal con secciones de servicios, información de la empresa y un formulario de contacto funcional.

## Uso

Abre `index.html` en tu navegador preferido. También puedes servir el sitio desde un servidor local con:

```bash
python -m http.server
```

Luego visita <http://localhost:8000>.

El formulario de contacto utiliza [Formspree](https://formspree.io) para el envío de mensajes. Reemplaza el endpoint en `script.js` por uno propio si deseas recibir los correos.

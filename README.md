# Lista de precios para ticket térmico 58mm

Aplicación web sencilla para crear una lista de precios y generar tickets listos para imprimir en impresoras térmicas de 58mm.

## Cómo usarla

1. Abre `index.html` en tu navegador.
2. Completa los datos del negocio.
3. Agrega productos con su precio.
4. Revisa la vista previa del ticket.
5. Presiona **Imprimir ticket** para enviar a la impresora térmica.

## Detalles

- Ajuste de ancho automático a 58mm para impresión.
- Guarda datos y productos en `localStorage` para no perder la información al recargar.
- Diseño responsivo para escritorio y móvil.
